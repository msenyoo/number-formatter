Number Formatter
=========

A small library that adds commas to numbers

## Installation

	'npm install number-formatter-msenyoo'

## Usage

	var numFormatter = require('number-formatter-msenyoo');
	var formattedNum = numFormatter(123456789);

Output should be '123,456,789'
